Краткое описание подхода (baseline → улучшения → финальное решение)
Сначала предобрабатывал данные.
Я использовал Наивный Байес в качестве baseline. Пробовал его калибровать.
Потом использовал KNN, SVM и RBF SVM (вышло 0.8) 
Бустинг(GradientBoosting, XGBoost, LightGBM, CatBoost) - но вышло хуже, чем Байес
Использовал стекинг (ГауссNB, KNN, random forest) - всё ещё не лучше Байеса
Но с добавлением градиентного бустинга стало лучше

Но в итоге домучал RBF SVM и на Kaggle вышло 0.804

Самое важное - это предобработка данных. С ней можно сидеть часами, но она позволяет сильно улучшить результаты. 
Особенно, если нет одного явно главного признака.
